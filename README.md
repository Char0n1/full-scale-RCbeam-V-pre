# 长安大学人工智能创新大赛
## 项目名称：足尺RC梁抗剪承载力预测
### 参赛赛道：
创新赛道
### 项目简介：
在桥梁设计中，传统规范中抗剪承载力公式往往不能较为准确地预测实际桥梁的承载能力，并且现有规范出于对结构安全的考量，在进行结构设计时会低估结构的承载能力，以增加结构安全冗余，然而更高的结构承载力意味着需要使用更多的材料，这就会造成经济损失。本项目旨在使用机器学习技术解决桥梁设计中对实际桥梁抗剪承载力的预测问题，由于机器学习的外推性能弱，且实际桥梁抗剪承载力数据量较少，所以无法准确预测实际桥梁的抗剪承载力，本项目通过梁的抗剪承载力具有尺寸效应的物理机理，将直接预测抗剪承载力问题转化为预测尺寸效应问题，较为精确地预测实际桥梁的抗剪承载力，平衡了实际工程中安全性和经济性。
